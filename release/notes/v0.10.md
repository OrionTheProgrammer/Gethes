# Gethes v0.10

## Online Snake Upgrade
- Added smoother and faster Snake input handling with improved direction buffering.
- Added online arena telemetry directly in-game:
  - live player count
  - room name
  - ping / sync age
  - current player rank
- Added room support for online matches:
  - `snake online room=<name>`
- Added online rival collision logic for more competitive runs.
- Added live side leaderboard updates for the arena snapshot.

## Network and Cloud Performance
- Optimized cloud client networking using persistent HTTP connections and keep-alive limits.
- Improved online arena sync cadence:
  - faster updates when state changes
  - lower network usage when idle
  - keepalive sync to preserve presence
- Extended arena payload and backend snapshot data with positional fields (`x`, `y`) and identity metadata.

## Distribution Improvements
- Build pipeline now supports bundling Syster runtime/model by default for release builds.
- Release workflow now installs Ollama on CI and enforces bundled Syster Core artifacts.
- Improved bundled runtime source detection in packaging scripts.

## Quality
- Added and updated automated tests for cloud sync, AWS cloud service, and Snake logic.
- Full test suite passes locally (`94 passed`).
