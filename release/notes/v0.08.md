# Gethes v0.08

## Resumen
Esta release consolida el salto de Gethes hacia una arquitectura con sincronizacion en la nube, mejora del entrenamiento de Syster y un pipeline de distribucion mucho mas estable.

## Cambios principales
- Se integro un backend cloud nativo para AWS (EC2) con servicio HTTP para telemetria y presencia.
- Se agrego persistencia de jugadores, perfiles, preferencias y puntajes en backend.
- Se amplio la sincronizacion para incluir datos de entrenamiento de Syster y muestras de feedback.
- Se extendio el backend Oracle para soportar perfil y feedback de Syster, manteniendo compatibilidad de despliegue.
- Se añadieron scripts de despliegue Linux para ambos escenarios: AWS y Oracle.

## Mejoras de calidad y estabilidad
- Se mejoro la robustez del flujo de release en GitHub Actions (correcciones de quoting en PowerShell).
- Se incorporo `PyInstaller` al entorno de CI para que la build de release sea reproducible.
- Se ajusto la compatibilidad de dependencias para evitar fallos de instalacion en pipelines.
- Se ampliaron pruebas automaticas para memoria de Syster, sincronizacion cloud y backend AWS.
