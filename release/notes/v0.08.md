# Gethes v0.08

## Summary
This release consolidates Gethes' transition to a cloud-capable architecture, stronger Syster training persistence, and a more reliable distribution pipeline.

## Main Changes

- Added an AWS-native cloud backend (EC2 + HTTP service) for telemetry and presence.
- Added backend persistence for players, profiles, preferences, and score data.
- Extended sync payloads to include Syster training summaries and feedback samples.
- Expanded Oracle backend compatibility for Syster profile and feedback ingestion.
- Added Linux deployment scripts for both AWS and Oracle scenarios.

## Quality and Stability

- Hardened the GitHub Actions release flow (PowerShell quoting and pipeline reliability fixes).
- Added `PyInstaller` to CI to keep release builds reproducible.
- Adjusted dependency compatibility to reduce install failures in CI/CD runs.
- Expanded automated tests for Syster memory, cloud sync, and AWS backend behavior.

## Distribution Artifacts

- `Gethes-Setup-v0.08.exe`
- `Gethes-v0.08-win64-portable.zip`
- `SHA256SUMS-v0.08.txt`

## Notes

- This version is infrastructure-focused and lays the base for upcoming gameplay and AI updates.
- Recommended for new users: installer package.
- Recommended for quick patching: in-game updater.
