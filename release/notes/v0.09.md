# Gethes v0.09

## Highlights
- Syster no longer sends automatic timed messages.
- Auto prompts on boot, idle, and command cadence are now disabled.
- Syster remains fully available when the player invokes it directly.

## Player Experience
- Removed unsolicited interruptions during gameplay and navigation.
- Preserved immersion by making Syster reactive instead of intrusive.

## Technical Notes
- Added a dedicated runtime gate for Syster auto-chat triggers.
- Kept manual Syster interaction paths unchanged.
