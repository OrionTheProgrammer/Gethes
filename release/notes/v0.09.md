# Gethes v0.09

## Highlights
- Live leaderboard panel added to the right side during gameplay.
- New cloud leaderboard categories: Snake, Roguelike, and Hangman.
- Syster replies are now fully asynchronous to avoid UI freezes.

## Gameplay & UI
- Snake now shows a real-time TOP panel while the match is running.
- Roguelike and Hangman received matching leaderboard support.
- Sidebar rendering is integrated into the in-game layout without breaking terminal style.

## Technical Notes
- Backend now exposes:
  - `GET /v1/leaderboard/snake`
  - `GET /v1/leaderboard/rogue`
  - `GET /v1/leaderboard/hangman`
- AWS SQLite schema auto-migrates Hangman leaderboard fields on startup.
- Cloud payload includes Hangman stats (`wins/games/best_errors_left`).
- Improved runtime responsiveness with non-blocking Syster inference pipeline.
