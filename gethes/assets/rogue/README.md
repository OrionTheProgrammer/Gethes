# Rogue Visual Assets

This folder contains the tile sprites used by Roguelike board rendering.

Supported files:

- `fog.png`
- `wall.png`
- `floor.png`
- `player.png`
- `exit.png`
- `gold.png`
- `potion.png`
- `relic.png`
- `trap.png`
- `enemy_g.png`
- `enemy_s.png`
- `enemy_w.png`
- `enemy_b.png`

Format recommendations:

- PNG with transparency
- Square images (`32x32` or `64x64`)
- Pixel-art friendly palette

You can replace these files directly and restart the game.
If a file is missing, Gethes falls back to icon/glyph rendering.
